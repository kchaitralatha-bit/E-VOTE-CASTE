# 🗳️ E-VOTE-CASTE (Online Voting System)

## 📌 Project Overview
E-VOTE-CASTE is a web-based online voting system that allows users to securely cast votes through a digital platform. The system includes authentication, vote storage, and result generation.

## 🚀 Features
- User Registration & Login
- Admin Panel
- Secure Voting Mechanism
- Real-time Result Display
- Database Integration

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Java (JSP/Servlet)
- Database: MySQL

## 🗄️ Database Setup
1. Import the SQL file into MySQL.
2. Update database credentials in configuration file.
3. Run the project on Apache Tomcat server.

## ▶️ How to Run
1. Start MySQL
2. Deploy project in Tomcat
3. Access via browser
